## CSS Image Fallbacks
To keep the website looking gorgeous before you add these files, we have implemented a **premium CSS fallback system**. 
If any image is missing, the site will automatically render a premium dark blue gradient box with a thin border and the appropriate descriptive label (e.g., "Showroom Exterior") in place of a broken image symbol.
